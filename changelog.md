# NXD FootballManager Changelog
All notable changes to this project will be documented in this file.
This project adheres to [Semantic Versioning](http://semver.org/).

## [2.3.5] – March 2026
- **Added:** Added support for Match Program

## [2.3.4] – March 2026
- **Fixed:** Fixed a bug in the coach model for storing coach <---> team associations
- **Fixed:** Fixed a bug in the player model for storing player <---> team associations
- **Changed:** Edit Layout for Coaches now has a larger gap between the two columns on the Basics view

## [2.3.3] – January 2026
- **Fixed:** Fixed a bug in the coach model for storing coach <---> team associations
- **Improved:** Country selection in Player, Coach, Official and Cheerleader Edit Views improved
- **Added:** Support to sort country selection in edit views

## [2.3.0] – January 2026
- **Improved:** Refactoring HtmlViews to Joomla 6.x coding standards (loading the model)
- **Added:** Added support for multi-citizenship for players, coaches, officials and cheerleaders

## [2.2.0] – January 2026
- **Added:** Added support for Matchball Sponsors
- **Added:** Implemented frontend view for Matchball Sponsors
- **Improved:** Component is now Joomla 6.x compatible without B/C Plugin requirement

## [2.1.6] – April 2025
- **Added:** Added inverted logo to the team model for NextGameModel

## [2.1.4] – March 2025
- **Added:** Implemented additional Fields: Team secondary colour, Team Inverted Logo, colour and secondary colour fields for Cheerleader Teams
- **Fixed:** Fixed some PHP Deprecated Warnings

## [2.1.2] – 2024
- **Fixed:** Model Filter was not working as expected
- **Improved:** Seasons select box now lists Seasons in descending order in Game Edit View for better UX
